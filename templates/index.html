How to use
  • Paste (⌘/Ctrl+V) or drag & drop files anywhere on the page — or type below and Send.
  • Text renders Markdown and HTML automatically; hit Show Raw on a message to see the source.
  • Each message has its own Copy and Delete. Press ⌘/Ctrl+C with nothing selected to copy the latest one.
  • Images, videos, and any other files are supported, several at once.
  • Open a private board by name from the top bar — it's protected by an authenticator-app code (TOTP), no password.
