Copy Paste Board

You can:
- Paste text or images here directly.
- Drag and drop images here.
- Type a message in the text area below and click "Send Message".
- Select an image file and click "Send Message".
- Copy a message by clicking the "Copy to Clipboard" button.
- Delete a message by clicking the "Delete" button.
- Delete all messages by clicking the "Delete All Messages" button.
- Copy the latest message by pressing #Copy Combination Key#.
- Messages are automatically updated every 5 seconds.
